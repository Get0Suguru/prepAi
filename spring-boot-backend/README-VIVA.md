# B.Tech 8th Sem - AI Smart Interview Prep (Spring Boot Guide)

This folder contains your **Real Project Backend**.

## How to set up local MySQL:
1. Install MySQL Server.
2. Create DB: `CREATE DATABASE prep_ai_db;`
3. Update `application.properties` with your MySQL username/password.

## How to Run:
1. Open this folder in **IntelliJ IDEA** or **Eclipse**.
2. Run `PrepAiApplication.java`.
3. The server will start on `http://localhost:8080`.
